Protocolo HTTP

HTTP vem de Hypertext Transfer Protocol(Protocolo de Transferência de Hipertexto). É um protocolo, uma forma de conversa entre duas máquinas, que permite transferir um hipertexto de um lado a outro.

Cliente e Servidor

O cliente e servidor se comunicam através de trocas de mensagens. As mensagens enviadas pelo cliente, geralmente um navegador da Web, são chamadas de requisições ou solicitações(requests), as mensagens que são enviadas pelo servidor como resposta são chamadas de respostas(responses).
